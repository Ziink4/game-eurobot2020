The Goal

Code an IA for the eurobot 2020 contest

  Rules

Read the eurobot 2020 contest rules

Encoder values are multiples of 0.1 mm

  Game Input

Input for one game turn
Line 1: 2 integers left_encoder1 and right_encoder1 for the current values of the robot 1 encoders.
Line 2: 2 integers left_encoder2 and right_encoder2 for the current values of the robot 2 encoders.
Output for one game turn
Line 1: 2 integers left_motor1 and right_motor2: motor outputs from -100 to 100 for robot 1 (percentage of maximum torque).
Line 2: 2 integers left_motor2 and right_motor2: motor outputs from -100 to 100 for robot 2 (percentage of maximum torque).
Line 3: 1 integer estimated_points: estimated points of the current match (for bonus computation)
Constraints
Response time for first turn ≤ 1000ms
Response time for one turn ≤ 50ms
