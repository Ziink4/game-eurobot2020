The Goal

Code an IA for the eurobot 2020 contest

  Rules

Read the eurobot 2020 contest rules

  Game Input

Input for one game turn
Line 1: 2 integers left_encoder and right_encoder for the current values of the robot encoders.
Output for one game turn
Line 1: 2 integers left_motor and right_motor: motor outputs from -100 to 100 (percentage of maximum torque).
Line 2: 1 integer estimated_points: estimated points of the current match (for bonus computation)
Constraints
Response time for first turn ≤ 1000ms
Response time for one turn ≤ 50ms
